Principais Habilidades
HTML
Linguagem de Marcação criada por Tim Berners-Lee. Tive o primeiro contato com HTML no inicio do curso técnico. Sempre gostei bastante de mexer com design interfaces. Tenho 2 anos de experiência com HTML
CSS
Sendo uma folha de estilos em cascatas, o CSS tem o objetivo de otimizar a aparência de um site. Assim como o HTML, tive o primeiro contato no inicio do curso técnico, tendo também em média 2 anos de experiência.
Java Script
Java Script foi a linguagem de programação que mais gostei desde que tive o primeiro contato com ela no curso técnico. De inicio, usei ela para fazer pequenos projetos, como pedra papel tesoura e um site que calcula o IMC. Após o fim do curso, me aprimorei e consegui realizar alguns projetos sozinhos, como um jogo da velha e um jogo da memória, além de pequenos usos em alguns mini websites. Depois que eu me aprimorei nela, eu parti para estudar algum framework.
Bootstrap
Bootstrap foi o primeiro framework que tive contato. Gostei muito da sua praticidade, mesmo atualmente não tendo tanto domínio dele. Meu objetivo é praticar e cada vez mais aumentar minha produtividade usando ele.
Vue.js
Vue.js foi o segundo framework que tive contato após o Bootstrap (Considerando o ReactJS como uma lib).
